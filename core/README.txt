This folder contains the implementation of the core architecture of
the agent (see docs/Architecture.pdf).  It contains two Eclipse Java
projects and one Visual Studio C# solution:

  agent - Eclipse Java project for the "server" part of the agent, which
          includes the collaboration manager, the relationship manager
	  and the semantic network

  Client - Visual Studio C# solution for "client" part of the agent, which
           includes the graphical UI

  discoRT - Eclipse Java project that supports the collaboration manager in
            the agent project

For more information contact rich@wpi.edu.
